Services & Status

System

Web Watcher scans your hardware to pick the best local AI models it can run. If you change your graphics card or memory, re-scan to switch to the models that fit your new hardware.
Loading system info…

AI model set

Bigger models understand you better; smaller ones use less memory and run lighter on your computer. Web Watcher keeps using your current model until a new one has finished downloading, then switches automatically.
Loading model sets…

Downloaded models

Delete any model you no longer use to free up disk space. The models currently in use are protected — switch to a different set first, then you can remove the old one.

The Watcher

Let The Watcher run my watches
One driver cycles through all your always-on watches like a person — choosing what to check next and telling you why in the dock — instead of each watch running on its own. Turn off to go back to starting watches individually.

Accessibility

Text size
Scales all text and UI elements. Preference is saved automatically.

Browser

Show browser window
When on, a Chromium window opens during each watch run so you can watch the agent navigate and interact with the page.
Stealth mode
Human-like mouse movements and timing to avoid bot detection. Disable only for simple trusted sites that don't need it.
Show agent cursor
Draws a red cursor in the agent's browser so you can watch where it clicks. Only visible when the browser is visible (turn off Headless above). Slightly reduces stealth, so keep it off for normal watching.
Connect Facebook
Opens a browser window so you can log into Facebook once. The saved session is reused by watches that have "Use saved login profile" enabled (e.g. Marketplace). Log in, then close the window.
Reset to a fresh install
Permanently erases everything personal — all watches, results, saved logins, chat history, and settings — and restarts the app like a brand-new install. The program itself (and the AI models) stay installed. This cannot be undone.

Console

A shell for the app folder (it runs windowless, so this is your console). Commands run on this machine — type carefully. cd persists between commands.
